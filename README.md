# Daily Tasks Manager 

## About

A simple to-do application for managing tasks with create, read, update, and delete functionalities. 

## Features

* Create tasks.
* Update details of exisitng task.
* Delete tasks.
* View created tasks.
* Can track status of tasks.
* Can track the priority level of a task.

## Tech Stacks

* Laravel 12
* Livewire
* Flux UI
* Tailwind Css
* MySQL (Database)
